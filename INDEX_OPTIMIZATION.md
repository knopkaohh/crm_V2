## 🔧 Скрипты для применения

### Windows

1. **`apply-optimization.ps1`**
   - PowerShell скрипт
   - Автоматическое применение
   - Запуск: `.\apply-optimization.ps1`

2. **`apply-optimization.bat`**
   - Batch файл
   - Для Command Prompt
   - Запуск: `apply-optimization.bat`

### Кроссплатформенные

3. **`backend/apply-indexes.js`**
   - Node.js скрипт
   - Прямое создание индексов
   - Запуск: `node backend/apply-indexes.js`

4. **`backend/package.json`** (скрипт)
   - NPM команда
   - Самый простой способ
   - Запуск: `cd backend && npm run apply-optimization`
