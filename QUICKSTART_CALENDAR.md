# 🚀 Быстрый старт: Производственный календарь

## 📋 Что нужно сделать

### Шаг 1: Применить миграцию базы данных

Откройте PowerShell в корневой папке проекта и выполните:

```powershell
.\apply-migration.ps1
```

Или вручную:

```powershell
cd backend
npx prisma db push
cd ..
```

### Шаг 2: Перезапустить backend

```powershell
cd backend
npm run dev
```

### Шаг 3: Убедиться, что frontend запущен

В другом терминале:

```powershell
cd frontend
npm run dev
```

## ✅ Готово!

Теперь вы можете:

1. **Открыть страницу заказов**: http://localhost:3000/orders
2. **Нажать на кнопку "Календарь"** (фиолетовая кнопка рядом с "Новый заказ")
3. **Перетащить заказ в колонку "В производстве"** - откроется модальное окно для указания сроков

## 🎯 Как использовать

### Добавить заказ в календарь:

1. Перейдите на страницу "Заказы"
2. Перетащите заказ в колонку "В производстве"
3. В открывшемся окне укажите даты для каждой позиции:
   - Дата начала производства
   - Дата окончания производства
4. Нажмите "Сохранить и продолжить"

### Просмотреть календарь:

1. Нажмите кнопку "Календарь" на странице заказов
2. Используйте стрелки ◀️ ▶️ для навигации по месяцам
3. Нажмите "Сегодня" для быстрого перехода к текущей дате
4. Кликните на день с позициями, чтобы увидеть детали

### Использовать фильтр:

1. На странице календаря нажмите кнопку "Наше производство"
2. Календарь покажет только позиции из материалов:
   - Сатин классический
   - Сатин премиум
   - Силикон
   - Нейлон

## 🎨 Цветовая схема материалов

- 🔵 **Синий** - Сатин классический
- 🟣 **Индиго** - Сатин премиум
- 🟣 **Фиолетовый** - Силикон
- 🌸 **Розовый** - Нейлон
- 🟢 **Зелёный** - Хлопок
- 🟡 **Жёлтый** - Жаккард
- 🟠 **Оранжевый** - Картон
- 🔷 **Бирюзовый** - ZIP-Lock пакет

## ❓ Возможные проблемы

### Миграция не применяется?

Убедитесь, что:
1. База данных запущена
2. В файле `backend/.env` правильно указан DATABASE_URL
3. У вас есть права на изменение схемы БД

### Календарь не загружается?

Проверьте:
1. Backend сервер запущен (порт 3001)
2. В консоли браузера нет ошибок (F12)
3. API доступен: http://localhost:3001/api/production-calendar

### Модальное окно не открывается?

1. Проверьте консоль браузера на ошибки
2. Убедитесь, что заказ имеет хотя бы одну позицию
3. Перезагрузите страницу
